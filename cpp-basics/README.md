# C++ Basics Assignment

This repository contains 8 tasks from my C++ college assignment.  
Each task is implemented as a separate `.cpp` file.

## Tasks

### 01_ascii_art.cpp
Prints an ASCII art image to the console.

### 02_string_length.cpp
Calculates and outputs the length of a string entered by the user.

### 03_rectangle.cpp
Prints a rectangle made of a character (`*`) with user-defined length and width.

### 04_diamond.cpp
Prints a diamond pattern made of `*` with a user-defined size.

### 05_snake_numbers.cpp
Prints numbers in a "snake" pattern inside a rectangle of user-defined size.

### 06_quadratic.cpp
Solves a quadratic equation `ax^2 + bx + c = 0` with user-provided coefficients.

### 07_ballistics.cpp
Calculates the distance a projectile will travel given initial velocity and launch angle.

### 08_ice_cream_sandwich.cpp
Checks whether a user-provided string is an "ice-cream sandwich" (pattern: edges repeated, middle different).

Example for any task:

```bash
g++ 01_ascii_art.cpp -o 01_ascii_art
./01_ascii_art
