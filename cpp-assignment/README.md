# C++ Assignment 1

This repository contains 5 tasks from my C++ college assignment.

Each task is a separate `.cpp` file. The tasks are explained below.

## Tasks

### task1.cpp
You are given two numbers (either entered by the user or defined as constants in the code). Find the sum of all even integers in the range between these numbers, inclusive.

### task2.cpp
You are given a number (either entered by the user or a constant in the code). Check whether it is negative or positive, and output the result to the console.

### task3.cpp
The code specifies a string of type `const char*`. Output the length of this string to the console.

### task4.cpp
You are given a number (either entered by the user or a constant in the code). Convert it to a `std::string`.

### task5.cpp
Find all the “lucky” tickets with numbers in the range from 000000 to 999999. A lucky ticket is one in which the sum of the digits of the first half equals the sum of the digits of the second half (e.g., 526265, 145226).

## How to compile and run
Example for task1:

```bash
g++ task1.cpp -o task1
./task1
